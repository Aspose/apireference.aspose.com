## Aspose.LLM
{{% alert color="primary" %}}
![Aspose.LLM for .NET Product Logo](aspose-llm-net.png)
Aspose.LLM is a .NET library for adding LLM and AI capabilities to your applications. It provides a simple, unified API for text and code generation, natural language understanding, chat sessions, and optional image input — no pipeline setup required. Load models via presets, run stateful chats, and use multimodal input when your model supports it. Built for .NET developers who want local inference without managing native binaries, Aspose.LLM handles model loading, session state, and inference so you can focus on your application logic.
