## MatchInfo.StartOfYear property

获取或设置雇主缴费上限重置的时间。某些计划的上限基于公司财政年度而非日历年度。若省略则假定为日历年度。仅使用月份和日期（MMDD），年份（YYYY）和时间将被忽略。

public string StartOfYear { get; set; }
