## Element.HasAttributeNS method

当此元素上指定了具有给定本地名称和命名空间 URI 的属性或该属性具有默认值时返回 true，否则返回 false。

```csharp
public bool HasAttributeNS(string namespaceURI, string localName)
```

| 参数 | 类型 | 描述 |
| --- | --- | --- |
| namespaceURI | String | 属性的命名空间 URI。 |
| localName | String | 属性的本地名称。 |

### 返回值

如果具有具有命名空间 URI 和本地名称的属性则返回 True；否则返回 false。
