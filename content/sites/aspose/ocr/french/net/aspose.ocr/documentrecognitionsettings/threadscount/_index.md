## DocumentRecognitionSettings.ThreadsCount property

Obtient ou définit le nombre de threads à traiter. Par défaut, 0 signifie que l'image sera traitée avec un nombre de threads égal à votre nombre de processeurs. ThreadsCount = 1 signifie que l'image sera traitée dans le thread principal.
