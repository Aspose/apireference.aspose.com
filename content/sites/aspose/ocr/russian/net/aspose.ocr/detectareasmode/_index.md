## DetectAreasMode enumeration

Определяет тип нейронной сети, используемой для обнаружения областей.

```csharp
public enum DetectAreasMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| NONE | `0` | Не определяет абзацы. Лучше для простого одноколоночного документа без картинок. |
| DOCUMENT | `1` | Обнаруживает абзацы, использует модель NN для документов. Лучше для многоколоночного документа, документа с картинками или с другими нетекстовыми объектами. |
| PHOTO | `2` | Обнаруживает абзацы, использует модель NN для фотографий. Лучше для изображения с большим количеством картинок и других нетекстовых объектов. |
| COMBINE | `3` | Обнаруживает абзацы с текстом, а затем использует другую модель NN для обнаружения областей внутри абзацев. Лучше для изображений со сложной структурой. |

### Примечания
