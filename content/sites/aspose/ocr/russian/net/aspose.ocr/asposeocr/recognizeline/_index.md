## RecognizeLine(string) {#recognizeline_1}

Распознает изображение, содержащее одну строку текста.  Автоматическая коррекция перекоса изображения не применяется. Поддерживает GIF, PNG, JPEG, BMP, TIFF, JFIF.

```csharp
public string RecognizeLine(string fullPath)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| fullPath | String | Путь к изображению. |

### Возвращаемое значение

Распознанный текст.
