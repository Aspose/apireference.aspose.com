## CharactersAllowedType enumeration

Καθορίζει τον τύπο των χαρακτήρων που επιτρέπεται για το αποτέλεσμα αναγνώρισης.

```csharp
public enum CharactersAllowedType
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| ALL | `0` | Αναγνώριση όλων των χαρακτήρων. |
| LATIN_ALPHABET | `1` | Αναγνώριση μόνο εάν ο χαρακτήρας είναι λατινικό αλφαβητικό γράμμα. |
| DIGITS | `2` | Αναγνώριση μόνο εάν ο χαρακτήρας είναι δεκαεξαδικό ψηφίο. |

### Παρατηρήσεις

Χρησιμοποιείται στο[`RecognitionSettings`](../recognitionsettings/) για να υποδείξετε ποιοι χαρακτήρες θα αναγνωριστούν.
