## DetectAreasMode enumeration

Καθορίζει τον τύπο του νευρωνικού δικτύου που χρησιμοποιείται για την ανίχνευση περιοχών.

```csharp
public enum DetectAreasMode
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| NONE | `0` | Δεν εντοπίζει παραγράφους. Καλύτερα για ένα απλό έγγραφο μιας στήλης χωρίς εικόνες. |
| DOCUMENT | `1` | Ανιχνεύει παραγράφους χρησιμοποιεί μοντέλο NN για έγγραφα. Καλύτερα για έγγραφα πολλών στηλών, έγγραφο με εικόνες ή με άλλα αντικείμενα εκτός κειμένου. |
| PHOTO | `2` | Ανιχνεύει παραγράφους χρησιμοποιεί μοντέλο NN για φωτογραφίες. Καλύτερα για εικόνα με πολλές εικόνες και άλλα αντικείμενα εκτός κειμένου. |
| COMBINE | `3` | Ανιχνεύει παραγράφους με κείμενο και στη συνέχεια χρησιμοποιεί άλλο μοντέλο NN για να ανιχνεύσει περιοχές μέσα στις παραγράφους. Καλύτερα για εικόνες με πολύπλοκη δομή. |
| TABLE | `4` | Ανιχνεύει κελιά με κείμενο. Προτιμώμενη λειτουργία για εικόνες με δομή πίνακα. |
| CURVED_TEXT | `5` | Ανιχνεύει γραμμές και αναγνωρίζει κείμενο σε καμπύλες εικόνες. Προτιμώμενη λειτουργία για φωτογραφίες σελίδων βιβλίων και περιοδικών. |

### Παρατηρήσεις

Χρησιμοποιείται στο[`RecognitionSettings`](../recognitionsettings/) για να καθορίσετε ποιον τύπο εικόνας θέλετε να αναγνωρίσετε.
