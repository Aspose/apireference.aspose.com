## ReceiptRecognitionSettings.ThreadsCount property

Λαμβάνει ή ορίζει τον αριθμό των νημάτων για επεξεργασία. Από προεπιλογή, το 0 σημαίνει ότι η εικόνα θα υποβληθεί σε επεξεργασία με τον αριθμό των νημάτων ίσο με τον αριθμό των επεξεργαστών σας. ThreadsCount = 1 σημαίνει ότι η εικόνα θα υποβληθεί σε επεξεργασία στο κύριο νήμα.

```csharp
public int ThreadsCount { set; }
```
