## BaseRecognitionSettings.SkewAngle property

Ottiene o imposta l'angolo in gradi per la rotazione dell'immagine.  L'impostazione di questo valore disabiliterà il[`AutoSkew`](../autoskew/) proprietà, in modo che la correzione automatica dell'inclinazione non venga applicata. Zero per impostazione predefinita.

```csharp
public float SkewAngle { set; }
```
