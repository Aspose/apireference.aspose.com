## DetectAreasMode enumeration

Determina il tipo di rete neurale utilizzata per il rilevamento delle aree.

```csharp
public enum DetectAreasMode
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| NONE | `0` | Non rileva i paragrafi. Meglio per un semplice documento a una colonna senza immagini. |
| DOCUMENT | `1` | Rileva i paragrafi utilizza il modello NN per i documenti. Ideale per documenti multicolonna, documenti con immagini o con altri oggetti non di testo. |
| PHOTO | `2` | Rileva i paragrafi utilizza il modello NN per le foto. Migliore per immagini con molte immagini e altri oggetti non di testo. |
| COMBINE | `3` | Rileva i paragrafi con testo e quindi utilizza un altro modello NN per rilevare le aree all'interno dei paragrafi. Migliore per immagini con struttura complessa. |
| TABLE | `4` | Rileva celle con testo. Modalità preferita per immagini con struttura a tabella. |
