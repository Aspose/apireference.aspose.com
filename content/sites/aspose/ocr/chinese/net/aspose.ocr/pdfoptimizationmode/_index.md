## PdfOptimizationMode enumeration

```csharp
public enum PdfOptimizationMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| NONE | `0` | 不优化 PDF 大小。 |
| MAXIMUM_QUALITY | `1` | 在保持最高图像质量的同时优化文件大小。 |
| HIGH_QUALITY | `2` | 以图像下采样为代价减小 PDF 文件大小。 |
| BALANCED | `3` | 下采样图像以平衡文件大小和图像质量。 |
| AGGRESSIVE | `4` | 在降低图像质量的代价下显著减小 PDF 文件大小。 |

### 备注
