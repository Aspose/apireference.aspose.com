## DocumentRecognitionSettings.DetectAreas property

获取或设置一个标志，指示是否应启用自动文本区域检测。启用 Document Structure Recognition 模块。这会增加处理图像的时间和内存消耗，但在复杂情况下提供更准确的结果。禁用（将其设置为 false）可加快图像处理，或在结构简单的图像情况下使用。

```csharp
public bool DetectAreas { set; }
```
