## ReceiptRecognitionSettings.ThreadsCount property

获取或设置用于处理的线程数。 默认情况下，0 表示将使用等于处理器数量的线程数处理图像。 ThreadsCount = 1 表示将在主线程中处理图像。

```csharp
public int ThreadsCount { set; }
```
