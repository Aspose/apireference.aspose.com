## Rectangle Inequality operator

测试两个 [`Rectangle`](../../rectangle) 结构的位置或大小是否不同。

```csharp
public static bool operator !=(Rectangle left, Rectangle right)
```

| Parameter | Type | 描述 |
| --- | --- | --- |
| left | Rectangle | 不等于运算符左侧的 [`Rectangle`](../../rectangle) 结构。 |
| right | Rectangle | 不等于运算符右侧的 [`Rectangle`](../../rectangle) 结构。 |

### 返回值

如果两个 [`Rectangle`](../../rectangle) 结构的 [`X`](../x)、[`Y`](../y)、[`Width`](../width) 或 [`Height`](../height) 属性中有任意不相等，则此运算符返回 true；否则返回 false。
