## BaseRecognitionSettings.SkewAngle property

Obtiene o establece el ángulo en grados para la rotación de la imagen.  Establecer este valor deshabilitará el[`AutoSkew`](../autoskew/) propiedad, por lo que no se aplica la corrección de desviación automática. Cero por defecto.

```csharp
public float SkewAngle { set; }
```
