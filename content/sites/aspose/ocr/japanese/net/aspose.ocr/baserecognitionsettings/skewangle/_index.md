## BaseRecognitionSettings.SkewAngle property

画像の回転角度を度単位で取得または設定します。 この値を設定すると、[`AutoSkew`](../autoskew/)自動スキュー補正が適用されないようにします. デフォルトではゼロです.

```csharp
public float SkewAngle { set; }
```
