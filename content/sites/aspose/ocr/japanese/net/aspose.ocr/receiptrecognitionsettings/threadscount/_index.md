## ReceiptRecognitionSettings.ThreadsCount property

処理するスレッドの数を取得または設定します。 デフォルトでは、0 はイメージがプロセッサの数と等しい数のスレッドで処理されることを意味します。 ThreadsCount = 1 は、イメージがメイン スレッドで処理されることを意味します。

public int ThreadsCount { set; }
