## CharactersAllowedType enumeration

認識結果に許可される文字の種類を決定します。

```csharp
public enum CharactersAllowedType
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| ALL | `0` | すべての文字を認識します。 |
| LATIN_ALPHABET | `1` | 文字がラテン アルファベット文字の場合のみ認識します。 |
| DIGITS | `2` | 文字が 16 進数の場合のみ認識します。 |
