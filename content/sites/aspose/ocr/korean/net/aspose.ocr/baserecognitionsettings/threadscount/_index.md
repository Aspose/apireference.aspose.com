## BaseRecognitionSettings.ThreadsCount property

처리할 스레드 수를 가져오거나 설정합니다. 기본적으로 0은 이미지가 프로세서 수와 동일한 스레드 수로 처리됨을 의미합니다. ThreadsCount = 1은 이미지가 기본 스레드에서 처리됨을 의미합니다.

```csharp
public int ThreadsCount { set; }
```
