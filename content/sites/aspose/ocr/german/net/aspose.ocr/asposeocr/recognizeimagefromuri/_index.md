## AsposeOcr.RecognizeImageFromUri method

Erkennt Text auf einem Bild, das von einem URI-Link bereitgestellt wird.

```csharp
public RecognitionResult RecognizeImageFromUri(string uri, RecognitionSettings settings = null)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| uri | String | URI-Link zum Bild. |
| settings | RecognitionSettings | Erkennungseinstellungen. |

### Rückgabewert

Der[`RecognitionResult`](../../recognitionresult/) Objekt mit Bilderkennungsergebnissen.

### Bemerkungen

Erkennt Bild mit der Fähigkeit zur Angabe[`RecognitionSettings`](../../recognitionsettings/) . Unterstützt GIF, PNG, JPEG, BMP, TIFF, JFIF.
