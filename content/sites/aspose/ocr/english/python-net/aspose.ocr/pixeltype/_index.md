## PixelType enumeration

Determines the type of the pixel data for image recognition.

## Members
| Member name | Description |
| :- | :- |
|BYTE|Sets type as 1 bit per pixel.|
|RGB|Sets type as rgb.|
|RGBA|Sets type as argb.|
|BGR|Sets type as bgr.|
