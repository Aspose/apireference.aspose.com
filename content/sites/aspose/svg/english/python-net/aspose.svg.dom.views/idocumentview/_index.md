## IDocumentView class

The DocumentView interface is implemented by Document objects in DOM implementations supporting DOM Views. It provides an attribute to retrieve the default view of a document.



The IDocumentView type exposes the following members:

### Properties
| Property | Description |
| :- | :- |
| [default_view](./aspose.svg.dom.views/idocumentview/default_view) | The default AbstractView for this Document, or null if none available. |
| [document](./aspose.svg.dom.views/idocumentview/document) | The source DocumentView of which this is an AbstractView. |
