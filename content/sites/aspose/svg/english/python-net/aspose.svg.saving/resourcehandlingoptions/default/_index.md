## default property


Gets or sets enum which represents default way of resources handling. Currently [`ResourceHandling.SAVE`](./aspose.svg.saving/resourcehandling#SAVE), [`ResourceHandling.IGNORE`](./aspose.svg.saving/resourcehandling#IGNORE) and [`ResourceHandling.EMBED`](./aspose.svg.saving/resourcehandling#EMBED) values are supported. Default value is [`ResourceHandling.SAVE`](./aspose.svg.saving/resourcehandling#SAVE).
