The **Aspose.Svg.Window**  namespace is related for
window object which represents a window
containing a DOM document. It includes classes and interfaces
related to active document address and browsing history.

### Classes
| Class | Description |
| :- | :- |
| [`IWindow`](./aspose.svg.window/iwindow) | The window object represents a window containing a DOM document. |
| [`IWindowEventHandlers`](./aspose.svg.window/iwindoweventhandlers) | Represents interface that must be inherited by window object |
| [`IWindowTimers`](./aspose.svg.window/iwindowtimers) | Allows authors to schedule timer-based callbacks. |
| [`Location`](./aspose.svg.window/location) | Location objects provide a representation of the address of the active document of their Document's browsing context, and allow the current entry of the browsing context's session history to be changed, by adding or replacing entries in the history object. |
